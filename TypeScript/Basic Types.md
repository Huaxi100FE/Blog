# typescript基本数据类型

![](images/types.png)
### 1. boolean类型
![](images/boolean.png)
```
let isDone: boolean = false;
```
### 2. number类型

![](images/number.jpg)
```
let decimal: number = 6;
let hex: number = 0xf00d;
let binary: number = 0b1010;
let octal: number = 0o744;
```
### 3. string类型

![](images/string.png)
```
let name: string = "bob";
name = "smith";
```
