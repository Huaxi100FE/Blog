# 代理 Proxies

### 1. 数组的解构赋值
+ 1.1基本用法
```
let a = 1;
let b = 2;
let c = 3;
```
等同于
```
let [a, b, c] = [1, 2, 3];
```
+ 1.2默认值

### 2. 对象的解构赋值
```
let { foo, bar } = { foo: "aaa", bar: "bbb" };
foo // "aaa"
bar // "bbb"
```
