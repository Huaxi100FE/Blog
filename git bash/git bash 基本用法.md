# git bash 基本用法

### 1. 初始化

```
$ git init
```
### 2. 克隆仓库

```
$ git clone 需要克隆的仓库地址
```
### 3. 拉取远程仓库代码

```
$ git pull origin master
```
### 4. 推送到远程仓库代码

```
$ git push origin master
```
### 5.查看远程仓库地址
```
$ git remote -v
```
### 6.撤消操作
```
$ git commit --amend
```
